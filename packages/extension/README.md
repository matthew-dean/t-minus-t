# TypeScript Minus TypeScript - VSCode Extension

VSCode/Cursor extension that lets you write TypeScript syntax in `.js` files, storing them as JavaScript with JSDoc comments.

## Installation

### From Source

1. Clone the repository
2. Run `pnpm install` from the root
3. Open the workspace in VSCode
4. Press `F5` to launch the extension in a new window

### From VSIX (when published)

1. Install the extension from the VSCode marketplace (when available)
2. Or install from a `.vsix` file: `code --install-extension typescript-minus-typescript-0.0.1.vsix`

## Usage

1. Open any `.js` file in VSCode/Cursor
2. Run the command `Typescript Minus TypeScript: Open as TypeScript` (or use the command palette)
3. Edit the file as if it were TypeScript
4. Save - your changes are automatically converted to JSDoc

## Configuration

- `tMinusTS.autoGenerateDts` (default: `false`) - Automatically generate `.d.ts` files when saving `.js` files

## Development

### Building

```bash
# From root
pnpm run build

# From extension directory
cd packages/extension
pnpm run build
pnpm run watch  # Watch mode
```

### Testing

The extension uses the `@t-minus-t/converter` package which has comprehensive tests. Run tests from the root:

```bash
pnpm test
```

### Linting

```bash
cd packages/extension
pnpm run lint
pnpm run lint:fix
```

## Architecture

This extension uses:
- `@t-minus-t/converter` - Core conversion logic (separate package)
- VSCode `TextDocumentContentProvider` API - For virtual document views
- TypeScript Compiler API - For `.d.ts` generation

## Project Structure

```
packages/extension/
  src/
    extension.ts          # Main entry point
    contentProvider.ts    # TextDocumentContentProvider
    dtsGenerator.ts      # .d.ts file generation
  dist/                   # Compiled output
```
